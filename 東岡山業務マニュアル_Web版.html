はじめに ― このマニュアルの使い方▶

サービス提供責任者および現場スタッフが、日々の業務を「同じ基準・同じ手順」で行えるようにまとめた業務手引きです。新人教育、応援時の確認、業務の見直しなど、迷ったときの“よりどころ”として使ってください。

運用ルール

  • 手順は標準です。利用者ごとの個別事項は「個別ケア表」を必ず確認してください。
  • 迷ったとき・判断に困ったときは、まず看護師または上長へ報告・相談します。
  • 「困ったら、迷ったら、まず人を呼ぶ」を全員の基本とします。
  • 改善点を見つけたら上長へ提案してください。マニュアルは育てていくものです。
POINT　本マニュアル内の「上長」はサービス提供責任者・施設長を指します（東岡山には副施設長は配置していません）。
第1章　1日の流れ（日勤・夜勤）▶

勤務帯ごとの基本的な動きです。時刻は目安。入居者の状態・行事・天候に合わせて柔軟に調整します。

第2章　日常ケアの手順▶

毎日くり返し行う基本ケアの手順です。安全と尊厳を最優先に、声かけを徹底します。

2-1　服薬

  1. お薬カレンダーを確認し、服薬が必要な入居者の薬を出す。
  2. 入居者の目の前で、名前・日付・いつの薬か（例：朝食後）を声に出して確認し、薬の封を切る。
  3. 口に薬が入った後は、必ず飲み込むまで目で確認する。
  4. 服薬後、服薬管理表へサイン（または押印）する。
  5. 内服薬の空袋入れに空袋を入れる（本人が拒否した薬も入れる）。

留意点

  • 血圧・甲状腺ホルモンなどの薬は、飲まないことで体調不良になることがあるため特に注意する。
  • 現在は必ずスプーンを使って職員が口まで運ぶ。
  • 口に入れたつもりでも服の上に落ちていることがあるため気をつける。
  • 服薬介助に使ったスプーンは、下膳食器と一緒に厨房へ出す。

担当の分担

  • 基本は看護師が内服薬をすべて対応する（不在時はヘルパーが対応）。
  • 起床・朝食前・眠前薬はヘルパーが対応する。
  • 昼・夕食後の内服薬確認は看護師が行う。
  • 眠前薬・インスリン・点眼薬は、必要になる時間の直前にナース室から出す（紛失防止）。
POINT　食後の下膳は服薬後に行います。看護師から服薬終了の報告を受けてから下膳します。

2-2　口腔ケア

  1. 感染予防のためプラスチック手袋を装着する。
  2. 個別ケア表に沿って歯ブラシ・口腔ケアスポンジ等を使用する（個人用のコップと歯ブラシを準備）。使用後は洗剤で洗う。
  3. 夕食後に個人用のコップを洗う。

2-3　体重測定

  1. 毎月第1水曜日に体重測定を行う（目安：昼食前の誘導時にホールで実施）。
  2. 体重計は特殊機械浴室に置いてある。
  3. 声かけをして測定する。
  4. 体重管理表に記入する（服の重さを500gとして引いた数字を記入）。
POINT　立位保持が困難な方は無理をせず、車いすを使用して測定します（車いすの重さを差し引く）。

2-4　介護記録の書き方

提供したサービスと利用者の状態を正しく残すための大切な業務です。短文でわかりやすく入力します。

  • 入居者の要望・ADL・状態に合った項目を使用して記入する。
  • 記録は1回の支援につき、いずれかの欄に短文で1か所入力すればよい。
  • 入浴時はバイタル（体温・血圧・脈拍）を記録する。

記録の例文

  • 起床・排泄：「体調確認を行う。特に体調不良はない。」
  • 就寝・口腔ケア：「声かけをしてパジャマへ一部介助で更衣する。」
  • 入浴：「見守りにて入浴する。」「体調不良のため看護師に相談し清拭対応とする。」
  • 服薬：「見守りで服薬介助を行う。」
  • バイタル記入例：「体温 36.4℃　血圧 121/67　脈 70」
第3章　訪問サービスの手順▶

訪問介護として提供する入浴・生活掃除・美容などの手順です。提供のたびに声かけを行い、変更があれば必ず報告します。

3-1　訪問入浴

準備

  1. 訪問予定を確認する。
  2. バイタルを確認する（生活支援記録表を確認。必要に応じて看護師に確認）。
  3. 衣服・入浴用品を準備する（着替え・シャンプー類・バスタオル1枚・フェイスタオル2〜3枚・入浴後の化粧水/軟膏/クリーム）。

入浴

  • 支援のたびに必ず声かけをする。
  • シャワー湯は自分の手で湯温を確認してからかける。
  • 全身の状態（皮膚・打撲など）を確認する。
  • 入浴後は湯冷めしないようしっかり拭き取る。
  • 異変があった際は看護師に連絡（不在時は他の職員を呼ぶ）。

記録・後片付け

  • 記録端末（カナエルタッチ）で記録する。
  • 着替え（洗濯物）を撮影し、洗濯業者へ出す袋へ入れる（ポケット内を必ず確認）。
  • バスタオルは本人専用のものを使用する（水虫予防）。
  • 最後の片付けで浴室のエアコン・電気を消し入口を施錠。換気扇は24時間つけたまま。
POINT　入浴中止（体調不良・外出など）は、生活支援表の特記事項に記入し、サービス提供責任者→担当ケアマネジャーへ報告します。

3-2　訪問生活掃除

  1. 訪問予定を確認する。
  2. 本人へ居室掃除の声かけをする。
  3. 居室掃除用品を準備する。
  4. 居室を掃除する（リネン交換・洗面台・ベッド周辺・トイレ・床など）。本人の物品を使う場合は了承を得る。
  5. 掃除道具を片付ける（雑巾は洗濯機で洗う）。
  6. 記録端末へ記録し、シーツ交換・清掃表へ記入する。

寝具（リース）について

  • 掛け布団・肌掛け布団・ベッドパッド・枕は予備が各2枚／月。超える交換は別途実費。
  • ラバーシーツの交換は2週間に1回（反対面を使用）。失禁跡がある場合は随時交換。
POINT　費用がかかるからといって入居者の処遇を下げることは決してしないでください。使用に差し支える汚染はこれまで通り交換します。

3-3　訪問美容

  1. 事務員が入居者へ訪問美容利用の意向を確認し、業者と連絡をとる。
  2. 毎月第2水曜日 9:00〜11:00 に訪問美容（業者）が来訪する。
  3. カット希望者はホールで待機する（居室の方は順に声かけして誘導）。
  4. パーマ・カラー希望者を先に誘導する。
POINT　現在は2階談話室を使用しています。
第4章　施設運営の手順▶

4-1　換気

現在は換気の時間帯は決めておらず、感染対策・環境衛生のため適宜対応しています。雨天時は全員で、天候に合わせて換気箇所を随時調整します。

4-2　備品・消耗品の管理

  1. 衛生用品・生活用品が半分程度になったら対応する。
  2. 施設の用品は所定の倉庫から出す。家族管理の場合はホワイトボードへ記入し、担当者がご家族へ連絡する。
  3. 倉庫から出したら備品提供表へ記入する。
  • 在庫が2〜3個と少なくなったら事務員へ報告する。
  • 補充が必要な物・家族依頼分はホワイトボードへ記入する。

4-3　病院受診時の対応

  1. 受診が必要と判断したら、看護師・上長に報告する。
  2. かかりつけ医に報告して指示をもらう（看護師）。
  3. 上長が家族へ連絡・相談する。
  4. 受診準備をする（付き添う職員の決定、車の準備など）。
  5. 出発→病院到着→受付→診察→精算。必要時は経過を上長へ報告。
  6. 帰所後、受診状況を看護師・上長に報告する。
  7. 上長から家族へ報告する。
  8. オプションサービスの用紙に記入し、受診記録を周知する。

持っていく物

  • 看護サマリー、各種保険証・受給者証・診察券（コピー可）、紹介状
  • 携帯電話、運転免許証、受診料・駐車料金、職員証、筆記用具
  • マスク（職員・入居者）、タオル、袋、必要に応じて傘・着替え・パッド類など

4-4　夜勤帯の緊急対応

夜間に転倒等の事故・急な体調不良が発生したときの手順です。落ち着いて順番に対応します。

  1. 呼吸停止・激痛・出血など一刻を争うときは、ためらわず119番通報し救急要請する。
  2. 施設長へ報告する（不在時は上長の順）。
  3. かかりつけ医に報告し指示をもらう。
  4. 家族へ連絡する。
  5. 救急搬送時はオンコールスタッフへ出勤要請する。

119番通報で伝える内容

  • 「救急です」と伝える／施設の種別／住所・電話番号（緊急連絡カード参照）
  • 利用者の状態・氏名・性別・年齢・病歴／搬送先の希望病院（看護サマリー参照）／対応者の名前
POINT　救急要請時は、青ファイル（緊急時情報・看護サマリー）を必ず持って付き添います。

4-5　ゴミ集め

  • 時間の目安は16:30〜18:00の間。
  • 対象：各フロアのホール・トイレ・手洗い場・事務所・洗濯場・浴室・居室など。
  • 不燃ごみ（食器類の割れ物）は新聞に包んで捨てる。

4-6　離床センサー（見守り機器）

  • 画面がアラート色（例：ピンク）になったら、対象の利用者の状況を確認する。
  • 離床アラートは「解除」操作をしない限り表示が戻らない（対応済みのチェック機構を兼ねる）。
  • 解除しないと次のアラートが判定できないため、必ず解除操作を行う。
POINT　機器の詳しい使用方法はメーカーの取扱説明書（別紙）を確認してください。

4-7　コスト管理の基本

  • 社内向け印刷は必要時のみ裏紙を使い、原則白黒・カラーは最小限。
  • 照明・エアコンは基準の目安（冷房27〜28℃、暖房18〜19℃）。電気代を受けている居室は強制設定にしない。
  • 充電器は満充電になったら電源を落とす。
  • 職員・入居者の紹介は紹介会社を使わず、全職員で紹介に協力する。
第5章　看取り期のケア▶

ご本人の苦痛をやわらげ、尊厳を守り、ご家族に寄り添う、特に大切な時期です。看護師・医師・上長と密に連携しチームで支えます。

5-1　基本姿勢

  • ご本人の安楽・尊厳を最優先にする。
  • ご家族の思いに寄り添い、こまめに状態を共有する。
  • 医師の指示・方針のもと、看護師を中心にチームで対応する。
  • 少しの変化も見逃さず、観察したことを記録し看護師・上長へ報告する。

5-2　観察と記録

記録表主な内容
看取り期バイタル表体温・血圧・脈拍・呼吸・酸素飽和度などの経過。
食事・水分摂取表食事量・水分摂取量の経過。
体位変換時間表褥瘡予防のための体位変換の時間・体位。
危篤時観察表危篤期に入った際の観察記録。
POINT　どの記録表をいつから使うかは医師・看護師の判断に従います。迷ったら必ず看護師へ確認してください。

5-3　苦痛の緩和と日常ケア

  • 口腔ケア：乾燥しやすいため、こまめに口腔内を潤し清潔を保つ。
  • 体位変換：褥瘡・苦痛予防のため、定められた時間で体位を変える。
  • 環境：室温・採光・においに配慮し、静かで落ち着ける環境を整える。
  • 声かけ：意識が低下していても、ふだんどおり優しく声をかける。

5-4　危篤時・急変時の対応

  1. 状態の変化に気づいたら、ただちに看護師・上長へ報告する。
  2. 医師・看護師の指示に従い、緩和を最優先に対応する。
  3. ご家族への連絡は上長・看護師が中心となって行う。
  4. 観察した内容を記録表に残す。
POINT　看取り期の方針は、事前にご本人・ご家族・医師と取り決めた内容に従います。必ず最新の方針を確認してください。
第6章　新人教育（OJT・育成計画）▶

新しく入った職員が安心して成長できるよう、計画的に育てます。教える側の心構えも大切にします。

6-1　育成の目標

  • 社会人・組織人・職業人（専門職）としての自覚を身につける。
  • 職場の理念を理解し、日々の実践につなげる。
  • 担当業務に必要な知識・技術を習得し、実践できる。
  • 「利用者主体のサービスとは何か」を考え、実践できる職員になる。

6-2　育成のスケジュール

時期区分到達目標
配属前研修OFF-JT理念・事業概要・就業規則などを学び、介護職としての基本的な知識・倫理観を身につける。
配属後〜1か月OJT日勤帯の業務を通じて利用者との関わり方・基本ケア（食事・入浴・排泄など）を学ぶ。
1か月振り返り面談現在の状況・課題について面談。悩みや課題を共有し不安を軽減する。
2〜3か月OJT基本ケアを実践。早出・遅出の一人立ち、夜勤業務に向けた準備を進める。

6-3　初日のOJTチェック項目

  • 施設概要・館内ラウンド・下駄箱・ロッカー・制服・暗証番号
  • 職員トイレ・健康チェック・駐車場・職員紹介
  • 経営理念カード・メール・勤怠・給与明細
  • 勤務表・就業規則・各種規程／日勤の流れ・顧客情報・1on1・バディ紹介
